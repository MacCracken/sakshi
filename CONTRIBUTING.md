# Contributing to Sakshi

Thank you for your interest in contributing to Sakshi.

## Development Setup

1. Install the Cyrius compiler (3.2.0+)
2. Clone this repository
3. `cyrius build` to compile
4. `cyrius test` to run tests

## Code Style

- Follow Cyrius conventions (see vidya reference)
- Zero heap allocation on the hot path
- Fixed buffers for all formatting
- Test every public function

## Pull Requests

- One feature per PR
- Include tests
- Update CHANGELOG.md
